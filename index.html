Introduction

Vous decouvrirez à travers la lecture de cette page l'outil de travail gratuit et open sourced nommé Git ainsi que toutes ses particularités qui en fait un outil essentiel pour tout travail de programmation.

Mais qu'est-ce que Git ?

Git est un outil de gestion de code source permettant un historique de version d'un programme ou tout autres fichiers de texte brut. Git organise les version en faisant des "noeuds" (appelés commits) contenant les informations des modifications depuis le dernier noeud créé. Ceci permet de plus de savoir plus facilement les modifications apportées à chacune des "versions" puisque chaque noeud doit être définit avec un résumé des modifications apportées. Finalement, chaque noeud contient de plus les informations de l'auteur de ce dernier, ainsi, on peut savoir facilement qui a fait quoi lorsqu'on travail en équipe.
